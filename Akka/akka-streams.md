# Akka Streams

* Akka Steams는 액터 기반 동시성 모델을 사용하는 Akka 툴킷 위에 Reactive Streams 사양을 구현한 것이다.
* Reactive Stream은 비동기식, 비블로킹, 이벤트 기반 데이터 처리에 중점을 두고 만들어졌다.
